# 🏢 HR System - Управление персоналом

**Современная система управления человеческими ресурсами** для корпораций, обеспечивающая эффективное управление сотрудниками, их ролями, зарплатами, документами и рабочими процессами.

## 🎯 Назначение системы

**HR System** - это корпоративная платформа для:
- 📊 **Управления персоналом** с четкой иерархией ролей
- 💰 **Контроля зарплат и бонусов** с ролевым доступом
- 📁 **Централизованного хранения документов** сотрудников
- 🏖️ **Управления отпусками и заявками** с системой одобрений
- 👥 **Командного взаимодействия** между сотрудниками и руководителями
- 🔐 **Безопасного доступа** с корпоративной аутентификацией

## 🚀 Технологический стек

### Frontend
- **React 19** - новейшая версия React с улучшенной производительностью
- **Next.js 15** - современный фреймворк с App Router и серверными компонентами
- **TypeScript** - строгая типизация для надежности кода
- **Tailwind CSS** - utility-first CSS для быстрой разработки UI
- **Responsive Design** - адаптивный дизайн для всех устройств

### Backend
- **Node.js** - высокопроизводительный JavaScript runtime
- **Express.js** - быстрый и минималистичный веб-фреймворк
- **TypeScript** - типизированный серверный код
- **JWT** - безопасная аутентификация пользователей
- **RESTful API** - стандартизированные API endpoints

### Архитектура
- **Monorepo** - единый репозиторий для frontend и backend
- **Yarn Workspaces** - эффективное управление зависимостями
- **Shared Types** - общие типы данных между слоями
- **Modular Design** - модульная архитектура для масштабируемости

## 👥 Система ролей и разрешений

### 🟢 **Employee (Сотрудник)**
**Цветовая схема: Зеленый**

**Возможности:**
- 👤 Просмотр и редактирование личного профиля
- 💰 Просмотр собственной зарплаты и бонусов
- 👥 Просмотр информации о команде и руководителе
- 📁 Загрузка и управление личными документами
- 🏖️ Подача заявок на отпуск и больничный
- 📱 Настройка социальных ссылок (Telegram, LinkedIn)
- 🖼️ Загрузка и управление аватаром профиля

**Ограничения:**
- ❌ Не может видеть зарплаты других сотрудников
- ❌ Не может управлять документами других пользователей
- ❌ Не может одобрять заявки на отпуск

### 🔵 **Manager/Team Lead (Руководитель команды)**
**Цветовая схема: Синий**

**Возможности:**
- ✅ Все возможности сотрудника
- 👥 Просмотр зарплат и бонусов всех членов команды
- 📊 Анализ эффективности команды
- ✅ Одобрение/отклонение заявок на отпуск от подчиненных
- 📝 Добавление комментариев к заявкам
- 🔍 Просмотр всех документов команды
- 📈 Управление командными процессами

**Команды:**
- **Frontend Team** → Team Lead: **Далер Алямов**
- **Backend Team** → Team Lead: **Денис**

## 🔌 API Endpoints

### 🔐 Аутентификация
```
POST /api/auth/login
Body: { email: string }
Response: { token: string, user: User }
```

### 👥 Пользователи
```
GET /api/users              # Список всех пользователей
GET /api/users/:id          # Информация о пользователе
GET /api/users/team/:team   # Члены команды
```

### 📁 Документы
```
GET /api/users/:id/documents     # Документы пользователя
POST /api/users/:id/documents    # Загрузка документа
Body: { name: string, type: string, file: File }
```

### 🏖️ Отпуска
```
GET /api/vacation/requests       # Заявки пользователя
GET /api/vacation/pending        # Ожидающие заявки (для менеджеров)
POST /api/vacation/request       # Создание заявки
PUT /api/vacation/request/:id/status  # Обновление статуса
```
